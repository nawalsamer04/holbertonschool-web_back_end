# ES6 Basics

This project introduces the fundamentals of ECMAScript 6 (ES6).

## Concepts Covered
- const and let
- Block scope
- Arrow functions
- Default parameters
- Rest and spread syntax
- Template literals
- Object property shorthand
- Iterators and for...of loops

## Requirements
- NodeJS 20.x
- Jest testing framework
- Babel
- ESLint
